Agenda

Conoce los contenidos que se van a compartir en el CaliDev.

  • 2:00 p.m.

    ¡Sólo funciona en mi PC!

    y en el de cualquiera

    Vagrant es una herramienta que permite crear ambientes de desarrollo ligeros y portables, solucionando el problema de algunos desarrolladores web que logran tener un producto que funciona sin problemas en un computador, pero tiene fallas en la máquina de otro miembro del equipo de desarrollo.

    Por Juan Manuel Álvarez

  • 3:00 p.m.

    Monitoring your API

    Ideas para monitorear un API

    Otro aspecto importante de desarrollar software es monitorearlo en producción, esta charla reúne recursos e ideas para monitorear un API usando los logs Nginx.

    Por Andrés F Vargas

  • 4:00 p.m.

    Docker

    First steps

    Primeros pasos en la plataforma para aplicaciones distribuidas de mayor crecimiento.

    Por Julián Camargo

  • 5:00 p.m.

    Keep calm and write reusable code in Android

    En la charla se hablará sobre el concepto de la reusabilidad de código y la forma en que lo estamos haciendo en Peewah para los proyectos en Android.

    Por Camilo Sacanamboy
